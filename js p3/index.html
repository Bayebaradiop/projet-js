Consigne

1) Lorsqu'on appuie sur entrée l'item saisi est ajouté comme un nouveau élément dans le bloc A

2) Lorsqu'on double-clique sur un élément dans le bloc A, ça le ramène dans le bloc B et l'élément est supprimé dans le bloc A

3) Lorsqu'on double-clique sur un élément dans le bloc B, ça le ramène dans le bloc A et l'élément est supprimé dans le bloc B

NB : Le champ de saisie ne doit pas être vide
